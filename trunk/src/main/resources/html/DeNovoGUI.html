DeNovoGUI

DeNovoGUI provides an easy to use graphical front end to the PepNovo+ algorithm for the de novo sequencing of MS/MS spectra.

To start the sequencing all you have to do is to select (i) the spectrum file(s) (in the mgf format), (ii) the settings to use, and (iii) the output folder for the sequencing results. During sequencing you will be informed of the progress and as soon as the process is complete the results will be displayed with the found sequences annotated on the spectra.

DeNovoGUI can also be run in command line mode. For more details see DeNovoCLI.
