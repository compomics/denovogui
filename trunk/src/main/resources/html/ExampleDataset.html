DeNovoGUI Example Dataset

The DeNovoGUI Example Dataset provides a realistic example of plant proteomics datasets. It consists of 30,289 MS/MS spectra of Arabidopsis thaliana tryptic peptides measured on an Orbitrap XL using CID fragmentation at the Salzburg University.
